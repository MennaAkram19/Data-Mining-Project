Breast Cancer Classification using k-NN

Course: Bioinformatics  
Student: Menna Akram Mahmoud  
ID: 2023023052

This project applies the k-Nearest Neighbors algorithm to classify breast cancer tumors as benign or malignant using the Wisconsin Diagnostic Breast Cancer dataset.

The project includes:
- Data exploration
- Data visualization
- Model training using kNN
- Model evaluation
